--Создать объект TestSuite автоматически на основе класса TestCase


Итак, мы создаем объект, который соответствует нашему тесту, в объекте содержится флаг.
Перед выполнением флаг должен находиться в состоянии ЛОЖЬ. Тестовый метод устанавливает флаг в состояние ИСТИНА.
Назовем тестовый класс WasRun, а флаг - wasRun.

Сделали, тест работает:
public void testApp()
    {
        WasRun test = new WasRun("testMethod");
        assertFalse(test.wasRun);
        test.run();
        assertTrue(test.wasRun);
    }

Теперь надо разделить обязанности, создадим суперкласс TestCase
